About Me
I'm an aspiring full-stack developer. I'm working hard to make sure that I have a solid understanding of both the front-end and the back-end. I enjoy gaining a deeper understanding of the languages I'm using by tackling new projects and overcoming the challenges that they present.
On the front-end, I have an understanding of HTML, CSS, and JavaScript. I've used Bootstrap and Materialize to design the front-end and give apps a professional look. I enjoy JavaScript the most on the front-end as it has a lot of utility and also a lot of challenges that are rewarding to overcome.
With the back-end, I've used Firebase to create a database on a website. I've also used node.js and express for the server side and linking into a database. I've used mySQL as well as Sequelize to connect into it; though, in my experience, creating my own ORM seems to work out better.
